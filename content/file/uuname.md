uuname命令用于显示全部的UUCP远端主机。

**语法格式：** uuname [参数]

**常用参数：**

-a 显示别名 -I 指定程序的配置文件 -l 显示本机名称 -v 显示版本信息 --help 显示帮助

**参考实例**

显示uucp主机名称：

```
[root@linuxcool ~]# uuname
```
