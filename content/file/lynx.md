lynx命令是终端上的纯文本模式的网页浏览器，没有JS引擎，不支持CSS排版、图形、音视频等多媒体信息。只能作为一个很有趣的小玩具。

**语法格式：** lynx [参数]

**常用参数：**

-case搜索字符串时区分大小写-ftp关闭ftp功能--color激活彩色模式--reload更新首页代理服务器的缓存--help 显示帮助

**参考实例**

访问www.linuxcool.com：

```
[root@linuxcool ~]# lynx www.linuxcool.com
```
