bzmore命令用于查看bzip2压缩过的文本文件的内容，当下一屏显示不下时可以实现分屏显示。

**语法格式：** bzmore [参数]

**参考实例**

分屏显示backup.tar.bz2压缩文件的内容：

```
[root@linuxcool ~]# bzmore backup.tar.bz2
```
